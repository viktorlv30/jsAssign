Window location

  • assign(url) - завантажуємо документ по даному url
  • reload([forceget]) - перезавантажити сторінку. Якщо forceget то завантажить сторінку без кешу
  • replace(url) - заміняє сторінку по url але не записує сторінку в історію
  • toString() - повертає url сторінки у вигляді стрічки

Window history

  • back() - преходить на попередню сторінку
  • forward - переходить на наступну сторінку
  • go(delta) - переходить на сторінку що знахиться в позиції дельта відносно нинішньої

Window document

Методи document
  • document.GetElementById('get-id')
  • document.getElementsByName('get-name');
  • document.getElementsByTagName('get-name');
  • document.GetElementsByClassName('get-class');
Методи самих елементів
  • getAttribute('class');
  • element.addEventListener('click', function(){ }); click, contextmenu, mouseover, mousedown, go
