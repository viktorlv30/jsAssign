Window location

  • assign(url) - завантажуємо документ по даному url
  • reload([forceget]) - перезавантажити сторінку. Якщо forceget то завантажить сторінку без кешу
  • replace(url) - заміняє сторінку по url але не записує сторінку в історію
  • toString() - повертає url сторінки у вигляді стрічки
